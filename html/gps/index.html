Reflection

  1. What is the purpose of HTML5 semantic tags? Which do you see most useful?

    2. The purpose of semantic tags are to just distinguish commonly used div class tags and to make them more readable(semantic) for organizing your HTML file. I think nav is the most useful semantic tag.

  2. What is the order the browser uses to display elements on a page?

    3. The order the browser uses to display elements is the order you write them out in your HTML file, unless you move elements around using CSS.

  3. What did you learn about Sublime in this challenge? Do you think you'll create more handy snippets at a later date?

    4. I learned how to make Sublime snippets which I think is one of the coolest things of Sublime, and yes I definitely will be making some of my own snippets later on

  4. What concept in this challenge was most difficult for you? How can you improve your understanding of this concept?

    5. There wasn't really any challenging concepts in this challenge for me. I didn't understand the background of HTML5 semantic tags before this challenge, also didn't know about snippets so those would've been challenging before hand but I understand concepts by reading out loud the structure of them and kind of orally teaching them to myself
